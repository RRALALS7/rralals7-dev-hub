# 🧠 Ideias de AI Tools

> Ferramentas com IA para estudar, criar, automatizar e ajudar pessoas.

---

## 9. Biblioteca de Prompts IA

**Descrição:** site com prompts organizados por categoria para devs, estudantes e criadores.

**Stack sugerida:** React, JSON, localStorage

**Dificuldade:** Fácil

**Recursos principais:**
- Busca por prompt
- Filtros por categoria
- Botão copiar
- Favoritos locais
- Tags de uso

**Roadmap:**
1. Criar layout com cards
2. Montar banco JSON de prompts
3. Implementar busca
4. Adicionar filtros
5. Criar favoritos

**Cuidados:** não incluir prompts para roubo, abuso, phishing ou automação maliciosa.

## 10. Bot Resumidor de Textos

**Descrição:** ferramenta que transforma textos longos em resumos claros e objetivos.

**Stack sugerida:** Node.js, Express, API de IA

**Dificuldade:** Média

**Recursos principais:**
- Resumo curto
- Resumo em tópicos
- Explicação simples
- Extração de pontos principais
- Limite de tamanho por texto

**Roadmap:**
1. Criar endpoint de resumo
2. Criar interface simples
3. Integrar IA
4. Adicionar modos de resumo
5. Salvar histórico opcional

**Cuidados:** avisar quando o conteúdo resumido pode perder detalhes importantes.

## 12. Gerador de README

**Descrição:** ferramenta que cria README profissional para projetos GitHub.

**Stack sugerida:** React, Markdown, API de IA opcional

**Dificuldade:** Fácil/Média

**Recursos principais:**
- Formulário do projeto
- Badges
- Instalação
- Uso
- Licença
- Preview em Markdown

**Roadmap:**
1. Criar formulário inicial
2. Criar template Markdown
3. Adicionar preview
4. Criar botão copiar
5. Adicionar exportação `.md`

**Cuidados:** não inventar informações falsas sobre o projeto.

## 13. Assistente de Estudos com IA

**Descrição:** app que ajuda estudantes com quiz, flashcards, resumos e cronogramas.

**Stack sugerida:** React, Node.js, SQLite, API de IA

**Dificuldade:** Média

**Recursos principais:**
- Criar flashcards
- Gerar quiz
- Explicar conteúdos
- Planejar rotina
- Revisão espaçada

**Roadmap:**
1. Criar cadastro local de matérias
2. Criar gerador de flashcards
3. Criar modo quiz
4. Adicionar cronograma
5. Criar revisão espaçada

**Cuidados:** não substituir professor, material oficial ou orientação escolar.
