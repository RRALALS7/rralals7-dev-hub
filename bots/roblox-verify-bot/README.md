# 🧱 RRALALS7 Roblox Verify Bot

> Bot Discord de verificação Roblox usando somente dados públicos.

---

## Como funciona

1. Usuário usa `/verify start username:SeuUsuario`
2. Bot gera um código público
3. Usuário coloca o código na descrição pública do perfil Roblox
4. Usuário usa `/verify check`
5. Bot verifica publicamente e entrega cargo opcional

---

## Recursos

- `/verify start`
- `/verify check`
- `/verify status`
- cargo verificado opcional
- dados locais em JSON

---

## Segurança

- Nunca pede senha
- Nunca pede cookie
- Nunca pede token
- Usa somente dados públicos
