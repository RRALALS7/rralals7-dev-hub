## Como funciona

1. Usuário usa `/verify start username:SeuUsuario`
2. Bot gera um código público
3. Usuário coloca o código na descrição pública do perfil Roblox
4. Usuário usa `/verify check`
5. Bot verifica publicamente e entrega cargo opcional
