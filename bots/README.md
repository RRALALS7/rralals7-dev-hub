## Stack padrão

- Node.js
- discord.js
- dotenv
- JSON local para dados simples
- Python em bots gamer/consulta quando fizer sentido

---

## Antes de rodar qualquer bot

1. Crie uma aplicação no Discord Developer Portal
2. Crie um bot
3. Copie o token com segurança
4. Configure `.env`
5. Registre comandos
6. Teste em servidor privado

---

## Segurança

- Nunca publique `.env`
- Nunca compartilhe token
- Use permissões mínimas
- Teste antes de colocar em servidor grande
- Não use os bots para spam, raid, bypass, roubo de conta ou coleta de dados privados

> RRALALS7 sempre ajuda.
