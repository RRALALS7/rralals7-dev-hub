## 27. Minecraft Server Status Bot

**Descrição:** bot que mostra status de servidores Minecraft.

**Stack sugerida:** Node.js, discord.js, API de status Minecraft

**Dificuldade:** Média

**Recursos principais:**
- Online/offline
- Quantidade de jogadores
- Ping
- Versão
- MOTD

**Roadmap:**
1. Criar comando `/mcstatus`
2. Consultar servidor
3. Mostrar embed
4. Adicionar cache
5. Criar monitoramento opcional

**Cuidados:** respeitar limites e evitar consultas excessivas.

## 30. Organizador de Torneios

**Descrição:** ferramenta para criar torneios, chaves e inscrições.

**Stack sugerida:** React, Node.js, SQLite

**Dificuldade:** Média/Alta

**Recursos principais:**
- Cadastro de times
- Inscrições
- Chaves simples
- Resultados
- Histórico de torneios

**Roadmap:**
1. Criar cadastro de torneio
2. Criar inscrição de jogadores/times
3. Gerar chaveamento
4. Registrar resultados
5. Criar página pública do torneio

**Cuidados:** criar regras claras e mecanismos contra edição indevida.
