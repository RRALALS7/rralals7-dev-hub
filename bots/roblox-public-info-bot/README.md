# 🧱 RRALALS7 Roblox Public Info Bot

> Bot Discord para mostrar informações públicas de perfis Roblox.

---

## Recursos

- `/roblox info`
- busca por username
- mostra ID público
- mostra display name
- mostra avatar público
- mostra link do perfil público

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/roblox info username:Roblox
```

---

## Segurança

- Usa somente dados públicos
- Não pede senha
- Não pede cookie
- Não pede token
- Não acessa conta de ninguém

> RRALALS7 sempre ajuda.
