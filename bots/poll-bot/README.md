# 🗳️ RRALALS7 Poll Bot

> Bot Discord simples para criar enquetes com reações.

---

## Recursos

- `/poll create`
- até 5 opções
- reações automáticas
- embed bonito

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/poll create pergunta:Qual jogo hoje? opcao1:Roblox opcao2:Minecraft opcao3:Outro
```

> RRALALS7 sempre ajuda.
