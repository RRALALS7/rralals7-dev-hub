# 📚 RRALALS7 Study Helper Bot

> Bot Discord simples para estudos, flashcards e quiz.

---

## Recursos

- `/study add`
- `/study list`
- `/study quiz`
- `/study remove`

---

## Ideia

O usuário cria flashcards com pergunta e resposta. Depois pode revisar em formato de quiz.

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/study add pergunta:O que é HTML? resposta:Linguagem de marcação para páginas web.
/study quiz
/study list
```

> RRALALS7 sempre ajuda.
