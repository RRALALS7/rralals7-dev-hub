# 🎁 RRALALS7 Giveaway Bot

> Bot Discord de sorteios simples com botão de participação.

---

## Status

```txt
✅ Testado e aprovado em ambiente real
```

Este bot já passou por teste prático e foi aprovado para entrar na lista de projetos validados do RRALALS7 Dev Hub.

---

## Recursos

- `/giveaway create`
- botão para participar
- sorteio automático por tempo
- `/giveaway reroll`
- `/giveaway list`

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/giveaway create premio:Nitro minutos:10 ganhadores:1
/giveaway list
/giveaway reroll message_id:ID_DA_MENSAGEM
```

---
