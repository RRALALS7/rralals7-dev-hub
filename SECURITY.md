# 🛡️ Política de Segurança

Este repositório é focado em ideias educativas, seguras e úteis.

---

## Reportando problemas

Se você encontrar uma ideia, texto ou exemplo que possa ser usado para abuso, abra uma issue explicando o problema de forma responsável.

Não inclua instruções perigosas, tokens, dados privados ou exemplos exploráveis.

---

## Conteúdo não permitido

Não aceitamos conteúdo relacionado a:

- roubo de contas
- phishing
- token grabber
- malware
- spam, raid ou flood
- bypass de ban
- selfbot abusivo
- exploração de usuários
- coleta de dados privados

---

## Boas práticas recomendadas

- Nunca publique tokens no GitHub
- Use `.env` para variáveis sensíveis
- Publique `.env.example` sem dados reais
- Valide entradas do usuário
- Crie limites de uso
- Use permissões mínimas necessárias
- Registre logs sem expor dados sensíveis

---

## Objetivo

Manter o RRALALS7 Dev Hub como um lugar seguro para aprender, criar e compartilhar ideias.
