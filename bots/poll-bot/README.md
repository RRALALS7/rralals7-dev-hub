> Bot Discord simples para criar enquetes com reações.

## Status

```txt
✅ Testado e aprovado em ambiente real
```

Este bot já passou por teste prático e foi aprovado para entrar na lista de projetos validados do RRALALS7 Dev Hub.

---

## Recursos

- `/poll create`
- até 5 opções
- reações automáticas
- embed bonito

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/poll create pergunta:Qual jogo hoje? opcao1:Roblox opcao2:Minecraft opcao3:Outro
```
