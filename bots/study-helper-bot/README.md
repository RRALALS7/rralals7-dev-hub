# 📚 RRALALS7 Study Helper Bot

> Bot Discord simples para estudos, flashcards e quiz.

---

## Status

```txt
✅ Testado e aprovado em ambiente real
```

Este bot já passou por teste prático e foi aprovado para entrar na lista de projetos validados do RRALALS7 Dev Hub.

---

## Recursos

- `/study add`
- `/study list`
- `/study quiz`
- `/study remove`

---

## Ideia

O usuário cria flashcards com pergunta e resposta. Depois pode revisar em formato de quiz.

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/study add pergunta:O que é HTML? resposta:Linguagem de marcação para páginas web.
/study quiz
/study list
```
