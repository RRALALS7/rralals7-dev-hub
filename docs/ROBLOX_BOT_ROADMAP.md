# 🧱 Roadmap — Roblox Utility Bot

> Caminho para criar bots e ferramentas Roblox usando somente dados públicos e métodos permitidos.

---

## Fase 1 — Ideia segura

Escolha um objetivo permitido:

- consulta de perfil público
- verificação de conta
- whitelist de comunidade
- status de experiências públicas
- painel de informações públicas

---

## Fase 2 — Entrada do usuário

- Username
- User ID
- Link de perfil
- ID da experiência

Sempre valide entradas antes de consultar APIs.

---

## Fase 3 — Consulta pública

- Usar APIs públicas quando disponíveis
- Criar tratamento de erro
- Criar cache
- Respeitar limites de requisição

---

## Fase 4 — Resposta bonita

- Avatar
- Nome
- ID
- Link público
- Status ou dados disponíveis

---

## Fase 5 — Segurança

Nunca pedir:

- senha
- cookie
- token
- código de autenticação
- dados privados

---

## Fase 6 — Documentação

Explique:

- quais dados são usados
- de onde vêm os dados
- quais limites existem
- que o projeto não acessa contas

---

## Checklist

- [ ] Usa apenas dados públicos
- [ ] Não pede credenciais
- [ ] Tem rate limit
- [ ] Tem tratamento de erro
- [ ] Tem README claro
