# 🧠 Visão Geral dos Sistemas — RRALALS7 Dev Hub

> Bots simples são ferramentas. Bots complexos são sistemas.

---

## Estrutura recomendada

```txt
rralals7-dev-hub/
├── bots/                 # bots simples e rápidos
├── complexos/            # sistemas avançados
├── architecture/         # explicação técnica
├── test-cases/           # casos de teste e bordas
├── production-templates/ # bases profissionais
├── docs/                 # documentação geral
└── site/                 # vitrine do projeto
```

---

## Três níveis de projeto

### Nível 1 — Bot simples

- fácil de rodar
- pouca configuração
- ideal para iniciantes

### Nível 2 — Bot complexo

- logs
- permissões
- regras configuráveis
- banco local ou SQLite
- módulos internos

### Nível 3 — Sistema completo

- bot + dashboard opcional
- banco robusto
- auditoria
- painel de configuração
- deploy profissional

---

## Regra do repo

Todo projeto precisa ser:

- útil
- seguro
- educativo
- configurável quando afetar membros
- claro para quem for usar
