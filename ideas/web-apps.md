## 23. Dashboard de Status de Bot

**Descrição:** painel web que mostra se bots estão online, ping, uptime e erros recentes.

**Stack sugerida:** React, Express, WebSocket, SQLite

**Dificuldade:** Média

**Recursos principais:**
- Status online/offline
- Ping
- Uptime
- Logs recentes
- Cards por bot

**Roadmap:**
1. Criar API de status
2. Criar cards no frontend
3. Atualizar status em tempo real
4. Registrar logs básicos
5. Adicionar autenticação simples

**Cuidados:** não expor tokens, IPs privados ou variáveis sensíveis.

## 24. Dev Toolbox Web

**Descrição:** site com ferramentas rápidas para devs.

**Stack sugerida:** React, JavaScript puro

**Dificuldade:** Fácil/Média

**Recursos principais:**
- Formatador JSON
- Base64 encode/decode
- Gerador UUID
- Conversor timestamp
- Contador de caracteres

**Roadmap:**
1. Criar layout com abas
2. Implementar formatador JSON
3. Implementar Base64
4. Implementar UUID/timestamp
5. Criar tema escuro

**Cuidados:** processar tudo localmente quando possível.
