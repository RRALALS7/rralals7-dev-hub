# 🧱 RRALALS7 Roblox Public Info Bot

> Bot Discord para mostrar informações públicas de perfis Roblox.

---

## Status

```txt
✅ Testado e aprovado em ambiente real
```

Este bot já passou por teste prático e foi aprovado para entrar na lista de projetos validados do RRALALS7 Dev Hub.

---

## Recursos

- `/roblox info`
- busca por username
- mostra ID público
- mostra display name
- mostra avatar público
- mostra link do perfil público

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/roblox info username:Roblox
```

---

## Segurança

- Usa somente dados públicos
- Não pede senha
- Não pede cookie
- Não pede token
- Não acessa conta de ninguém
- Ideal para consulta informativa em comunidades Roblox
