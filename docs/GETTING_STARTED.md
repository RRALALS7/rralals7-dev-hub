# 🚀 Começando no RRALALS7 Dev Hub

> Guia rápido para transformar uma ideia em projeto real.

---

## 1. Escolha uma ideia

Abra a pasta [`ideas`](../ideas) e escolha uma categoria:

- Discord Bots
- AI Tools
- Roblox Utils
- Security Tools
- Web Apps
- Game Tools

---

## 2. Copie o template

Use um dos modelos:

- [`templates/idea-template.md`](../templates/idea-template.md)
- [`templates/project-roadmap.md`](../templates/project-roadmap.md)

---

## 3. Defina o MVP

MVP é a primeira versão simples do projeto.

Exemplo:

```md
Meu bot precisa primeiro:
- responder um comando
- salvar configuração
- mostrar resposta bonita
```

---

## 4. Crie o repositório

Sugestão de estrutura:

```txt
meu-projeto/
├── README.md
├── src/
├── docs/
├── .env.example
├── package.json
└── LICENSE
```

---

## 5. Segurança básica

Antes de publicar:

- não poste tokens
- não poste senhas
- não poste cookies
- crie `.env.example`
- use `.gitignore`
- explique limites do projeto

---

## 6. Publique e evolua

Comece pequeno, publique, melhore e documente.

> RRALALS7 sempre ajuda.
