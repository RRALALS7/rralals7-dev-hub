# 🛡️ RRALALS7 Anti Scam Bot

> Bot Discord defensivo para checar links suspeitos.

---

## Recursos

- `/checklink`
- verificação manual de link
- whitelist local
- blacklist local
- resultado em embed
- uso opcional de Google Safe Browsing

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/checklink url:https://github.com
```

---

## Segurança

- O bot não abre links
- Resultado é indicação de risco, não sentença absoluta
- Não use para perseguir usuários
- Ideal para proteção de comunidade

> RRALALS7 sempre ajuda.
