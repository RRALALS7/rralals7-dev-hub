# 🧠 RRALALS7 AI Moderator Bot

> Bot Discord com respostas seguras, FAQ e ajuda para comunidade.

---

## Recursos

- `/ask`
- `/faq add`
- `/faq list`
- `/faq remove`
- filtro simples contra pedidos perigosos
- respostas baseadas em FAQ local

---

## Observação

Esta versão não usa API externa de IA ainda. Ela funciona como um moderador inteligente simples com FAQ local e respostas seguras. Depois pode ser conectada com Gemini, Groq ou outro provedor.

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```
