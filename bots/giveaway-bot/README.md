# 🎁 RRALALS7 Giveaway Bot

> Bot Discord de sorteios simples com botão de participação.

---

## Recursos

- `/giveaway create`
- botão para participar
- sorteio automático por tempo
- `/giveaway reroll`
- `/giveaway list`

---

## Instalação

```bash
npm install
cp .env.example .env
npm run register
npm start
```

---

## Uso

```txt
/giveaway create premio:Nitro minutos:10 ganhadores:1
/giveaway list
/giveaway reroll message_id:ID_DA_MENSAGEM
```

---

## Segurança

- Sem dinheiro real
- Regras claras no servidor
- Staff deve revisar sorteios importantes

> RRALALS7 sempre ajuda.
