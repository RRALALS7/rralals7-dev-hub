## Instalacao

```bash
npm install
cp .env.example .env
npm run register
npm start
```

## Uso responsavel

- A moeda e ficticia e nao tem valor real
- Nao usar para apostas, venda, troca real ou dinheiro real
- Projeto feito para diversao e aprendizado

> RRALALS7 sempre ajuda.
