# 🤝 Contribuindo com o RRALALS7 Dev Hub

Obrigado por querer contribuir!

Este projeto existe para ajudar devs a encontrarem ideias seguras, úteis e educativas.

---

## ✅ O que é bem-vindo

- Ideias de bots úteis
- Ferramentas de IA educativas
- Utilitários gamer legalizados
- Projetos web para portfólio
- Checklists de segurança defensiva
- Templates de README e roadmap
- Melhorias na organização do repositório

---

## ❌ O que não será aceito

Não envie conteúdo relacionado a:

- roubo de contas
- token grabber
- phishing
- spam, raid ou flood
- selfbot abusivo
- bypass de ban
- coleta de dados privados
- malware
- exploração de plataformas ou usuários
- automações que violem regras de serviços

---

## 🧩 Como sugerir uma ideia

1. Abra uma issue
2. Use um título claro
3. Explique o objetivo
4. Diga a stack sugerida
5. Inclua cuidados de segurança

Modelo recomendado:

```md
## Nome da ideia

## Descrição

## Stack sugerida

## Dificuldade

## Recursos principais

## Cuidados de segurança
```

---

## 🛠️ Como enviar Pull Request

1. Faça um fork
2. Crie uma branch
3. Adicione ou melhore arquivos
4. Revise segurança e clareza
5. Abra um Pull Request
